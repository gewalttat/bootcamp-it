# 📦 Модульная архитектура приложения "WB Junior Clone"

## Структура проекта

```
lesson-5/client/

├── script.js            # ❌ DEPRECATED - старый файл (можно удалить)
├── modules/
│   ├── api.js          # 🌐 Работа с сервером и API
│   ├── storage.js      # 💾 Управление localStorage
│   ├── cart.js         # 🛒 Логика корзины
│   ├── catalog.js      # 📺 Отображение товаров
│   ├── filter.js       # 🔍 Фильтрация и сортировка
│   ├── header.js       # 📋 Хедер и поиск
│   └── main.js         # 🚀 Инициализация приложения
```

## Описание модулей

### 1️⃣ **api.js** - API и загрузка данных
Отвечает за коммуникацию с сервером.

**Функции:**
- `loadGoods()` - загрузить товары с сервера
- `getGoodById(id)` - получить товар по ID
- `getAllGoods()` - получить все товары
- `getGoodsByCategory(category)` - товары по категории
- `searchGoods(query)` - поиск товаров

### 2️⃣ **storage.js** - Локальное хранилище
Работает с localStorage для сохранения данных браузером.

**Функции:**
- `saveCart(cartItems)` - сохранить корзину
- `loadCart()` - загрузить корзину
- `clearCart()` - очистить корзину
- `saveFavorites(favorites)` - сохранить избранное
- `loadFavorites()` - загрузить избранное

### 3️⃣ **cart.js** - Управление корзиной
Логика работы с корзиной товаров.

**Функции:**
- `initCart()` - инициализировать корзину
- `addToCart(id)` - добавить товар
- `removeFromCart(index)` - удалить товар
- `clearAllCart()` - очистить всё
- `getCartCount()` - количество товаров
- `getCartTotal()` - сумма товаров
- `getCartItems()` - все товары корзины
- `updateCartCounter()` - обновить счетчик в UI

### 4️⃣ **catalog.js** - Каталог товаров
Отображение товаров и модальные окна.

**Функции:**
- `renderGoods(data)` - отрисовать товары
- `openModal(id)` - открыть модаль товара
- `closeModal()` - закрыть модаль
- `addToCartAndClose(id)` - добавить и закрыть

### 5️⃣ **filter.js** - Фильтрация и сортировка
Фильтры и сортировка товаров.

**Функции:**
- `filterCategory(categoryName)` - фильтр по категории
- `sortByPrice(order)` - сортировка по цене
- `sortByRating()` - сортировка по рейтингу
- `sortByDiscount()` - сортировка по скидке
- `filterByPrice(min, max)` - фильтр диапазон цен
- `filterByRating(minRating)` - фильтр по минрейтингу

### 6️⃣ **header.js** - Хедер и навигация
Функции хедера, поиска и корзины.

**Функции:**
- `initSearch()` - инициализировать поиск
- `openCart()` - открыть корзину
- `closeCart()` - закрыть корзину
- `checkoutOrder()` - оформить заказ

### 7️⃣ **admin.js** - Управление товарами ⭐ НОВОЕ
Создание, редактирование и удаление товаров (через сервер).

**Функции:**
- `initAdmin()` - инициализировать админ-панель
- `openAddProductModal()` - открыть форму добавления
- `closeAddProductModal()` - закрыть форму
- `addProductToServer(product)` - отправить товар на сервер (POST)
- `deleteProductFromServer(id)` - удалить товар с сервера (DELETE)
- `validateProduct(product)` - проверить корректность данных
- `fillRandomImage()` - генерировать случайную картинку

### 8️⃣ **main.js** - Главная инициализация
Точка входа приложения.

**Функции:**
- `initApp()` - инициализировать все модули

## Порядок загрузки модулей в HTML

Модули загружаются в правильном порядке, чтобы избежать зависимостей:

```html
<!-- Сначала базовые модули (используются другими) -->
<script src="modules/api.js"></script>
<script src="modules/storage.js"></script>
<script src="modules/cart.js"></script>

<!-- Потом UI модули -->
<script src="modules/catalog.js"></script>
<script src="modules/filter.js"></script>
<script src="modules/header.js"></script>

<!-- Админ модуль -->
<script src="modules/admin.js"></script>

<!-- И наконец main - который всё инициализирует -->
<script src="modules/main.js"></script>
```

## Как это работает

1. **При загрузке страницы:**
   - main.js запускает `initApp()`
   - Инициализируется корзина (загружается из localStorage)
   - Загружаются товары с сервера (api.js)
   - Отрисовываются товары (catalog.js)
   - Инициализируется поиск (header.js)
   - Инициализируется админ-панель (admin.js)

2. **При клике на товар:**
   - `openModal()` из catalog.js
   - Показывает детали товара
   - Можно добавить в корзину

3. **При добавлении в корзину:**
   - `addToCart()` из cart.js
   - Сохраняется в localStorage (storage.js)
   - Обновляется счетчик

4. **При открытии корзины:**
   - `openCart()` из header.js
   - Загружает товары из `getCartItems()` (cart.js)
   - Позволяет удалять товары или оформить заказ

5. **⭐ При добавлении НОВЫХ товаров:** (НОВОЕ!)
   - Клик на кнопку "➕ Добавить товар" в хедере
   - Открывается модаль с формой (admin.js)
   - Заполняем поля: категория, бренд, название, цена, рейтинг, картинка
   - Клик "✓ Добавить товар"
   - `addProductToServer()` отправляет POST запрос на http://localhost:3000/api/products
   - Сервер сохраняет в db.json
   - Товары перезагружаются
   - Новый товар появляется на витрине!

## 🚀 Как запустить приложение

### Требования:
- Node.js + npm (для сервера)
- Браузер с поддержкой ES6

### Шаги:

1. **Запустить сервер:**
   ```bash
   cd lesson-5/server
   npm install
   npm start
   # Сервер будет на http://localhost:3000
   ```

2. **Запустить клиент:**
   ```bash
   cd lesson-5/client
   python3 -m http.server 8000
   # или используй любой другой HTTP сервер
   # Открой http://localhost:8000 в браузере
   ```

3. **Готово!** Приложение работает 🎉
